Several hotkeys have been set to simplify your navigation.

  • J/K: preselect previus/following row
  • S: select/unselect a preselected row

  • N: add a new group

  • ENTER: edit/mass edit selected row(s)

  • SUPPR/DELETE: massive delete operation on selected row(s)
  • L: massive delete operation on selected row(s)
  • X: massive export operation on selected row(s)
  • P: massive toggle lock operation on selected row(s)

  • ESCAPE: (in order)
    • hide the modal
    • unfocus the search field
    • unselect selected row(s)
